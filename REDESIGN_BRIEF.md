# Practical Library Redesign Brief

## Vision
Transform practicallibrary.com into an Apple-level quality educational platform that teaches non-technical people to leverage AI tools to transform their business.

## Core Message
**"If you can send an email, you can use Claude Code. If you can use Claude Code, you can change the way you do business."**

## Target Audience
- Non-technical business owners and professionals
- People who can use email but think "coding" is beyond them
- Entrepreneurs looking to leverage AI without hiring developers

## Design Philosophy
- **Apple-level quality** - Clean, elegant, premium feel
- **Non-intimidating** - Warm, welcoming, accessible
- **Educational-first** - Content that teaches, not just sells
- **Hormozi-style value** - Provide massive free value upfront

## Monetization Strategy (Subtle)
- Free: All educational content, guides, tutorials
- Upsell: Setup services for Claude Code and Clawdbot
- Premium: Live workshops, personalized coaching

## Key Pages
1. **Home** - Hero with clear value prop, learning paths, social proof
2. **Guides** - Step-by-step tutorials with screenshots
3. **Tools** - Essential vs Power tools categorization
4. **Workshops** - Live training offerings
5. **Setup Services** - Done-for-you Claude Code/Clawdbot setup

## Design Requirements
- Modern, clean typography
- Generous whitespace
- Subtle animations (not flashy)
- Mobile-first responsive
- Fast loading (static where possible)
- Accessible (WCAG compliant)

## Tech Stack (Keep)
- Next.js 15
- Tailwind CSS
- Vercel deployment

## Inspiration
- Apple.com (clean, confident)
- Linear.app (modern SaaS)
- Notion (educational, approachable)
- Stripe (developer-friendly but elegant)

## Content Tone
- Encouraging, not condescending
- "You can do this" energy
- Practical, not theoretical
- Results-focused
